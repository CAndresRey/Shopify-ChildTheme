## Getting started

1. Clone:

```sh
git clone
```

2. Add news changes:

```sh
git add .
```

3. Committed changes

```sh
git commit -m "contribute commit"
```

4. Pushed changes:

```sh
git push -u origin main
```

7. Install Shopify CLI
6. Create development store in Shopify

5. Check live changes on:

```sh
shopify theme login --store=<mystore.myshopify.com>
```

```sh
shopify theme serve
```
